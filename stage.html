GEIREC – Larmor-Plage Mars – Juin 2026 12 semaines

Contexte

GEIREC est un cabinet d'expertise comptable breton de près de 400 collaborateurs répartis sur six sites. Mon stage s'est déroulé au sein du pôle Data sur le site de Larmor-Plage. L'objectif était de transformer des données comptables hétérogènes en outils d'aide à la décision fiables et exploitables pour les clients du cabinet.

Missions

Le stage a été structuré autour de quatre axes de travail

Tableaux de bord Power BI

Conception et maintenance de dashboards financiers interactifs pour le pilotage des clients.

Fiabilisation des données

Contrôle qualité, détection d'anomalies et nettoyage des données sources avant exploitation.

Architecture Microsoft Fabric

Structuration des flux de données via une architecture en couches Bronze / Silver / Gold dans le cloud.

Automatisation

Remplacement de traitements manuels Excel par des solutions Power Query, VBA et Python.

Projets réalisés

01

Intégration budgétaire – Groupe de restauration rapide

Intégration de budgets prévisionnels multi-exercices dans un tableau de bord Power BI existant. Développement de mesures DAX pour calculer les écarts Réel / Budget et les cumuls Year-To-Date sur 25 restaurants.

Power BI DAX Power Query
02

Geirec Académie – Migration vers Microsoft Fabric

Refonte complète de l'infrastructure data de la plateforme de formation interne. Migration depuis des fichiers Excel locaux vers un Lakehouse Fabric centralisé avec import de 22 fichiers JSON, modélisation relationnelle et déploiement d'un modèle sémantique partagé.

Microsoft Fabric JSON Power Query DAX
03

Tableau de bord financier – Restauration traditionnelle

Construction d'un tableau de bord complet à partir de l'API Pennylane pour deux établissements avec exercice comptable décalé (décembre–novembre). Développement de plus d'une centaine de mesures DAX incluant la segmentation solide / liquide et le suivi budgétaire.

Power BI API Pennylane DAX Python
04

Automatisation comptable – Client opticien

Traitement automatisé d'exports d'un logiciel de caisse via neuf requêtes Power Query liées. Génération automatique d'une feuille CA reclassée, d'un journal des ventes et d'un journal des encaissements conformes aux normes comptables.

Power Query Excel
05

Balances mensuelles automatiques – Client bricolage

Transformation d'un grand livre brut CSV en balances mensuelles structurées via Power Query, avec génération automatique des feuilles Excel par une macro VBA.

Power Query VBA Excel
06

Tableau de bord de pilotage – Groupe dentaire

Maquettage d'un dashboard de pilotage avec données fictives, pipeline d'extraction Pennylane pour deux entités juridiques, pseudonymisation des données patients et mise en place de la Row-Level Security (RLS).

Power BI Microsoft Fabric Python RLS

Outils et technologies

Tous les projets s'appuient sur une base technique mobilisée tout au long du stage.

Power BI Desktop Microsoft Fabric DAX Power Query (M) Python (pandas / PySpark) VBA Excel JSON Lakehouse

Bilan

Apports techniques

Montée en compétence sur Power BI, DAX et Power Query avec plus d'une centaine de mesures développées. Première expérience significative sur Microsoft Fabric et les architectures cloud de données.

Apports pour le cabinet

Réduction des manipulations manuelles, fiabilisation des données et fourniture aux clients d'indicateurs accessibles en temps réel pour le pilotage de leur activité.

Compétences acquises

Compréhension des mécanismes comptables (SIG, PCG, exercices décalés), collaboration inter-sites et gestion de plusieurs projets en parallèle dans un environnement professionnel.

Limites identifiées

Qualité et disponibilité des données parfois contraignantes, développements réalisés sur données fictives en attente d'accès aux systèmes réels, profondeur d'approfondissement limitée par la diversité des projets.
